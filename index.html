First Time Setup

If this is your first time using this reporting tool, do the following:

  • Open this link and copy the report into your drive.
  • Name the document whatever you like, but don't change the sheet name from Sheet1.
  • Change the data out for the client you are working on.
  • When ready, share the document so that anyone with the link can edit it.
